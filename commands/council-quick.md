# /council-quick - 신속 위원회

Stage 2(교차 평가)를 생략하고 빠르게 다양한 관점을 수집합니다.

## 사용법
```
/council-quick <질문>
```

## 실행 지침

$ARGUMENTS 를 사용자 질문으로 처리하세요.

---

## 2단계 신속 프로세스

### Stage 1: 독립적 의견 수집

**1. Gemini 위원:**
```bash
gemini "다음 질문에 대해 전문적인 견해를 제시해 주세요.

질문: {사용자질문}

답변 형식:
1. 핵심 요약
2. 상세 설명
3. 근거" --yolo
```

**2. Codex 위원:**
```bash
codex exec "다음 질문에 대해 전문적인 견해를 제시해 주세요.

질문: {사용자질문}

답변 형식:
1. 핵심 요약
2. 상세 설명
3. 근거"
```

**3. Claude 위원 (의장):**
동일한 형식으로 자신의 의견을 생성합니다.

### Stage 3: 즉시 종합 (Stage 2 생략)

의장이 모든 응답을 종합하여 결론 작성.

---
